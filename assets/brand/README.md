# Noval brand assets

These are the project author's selected raster assets. Keep their pixels,
colors, dimensions, and filenames unchanged unless a new selected package
replaces them.

## Logo

| File | Intended use |
|---|---|
| `logo/noval-logo-primary.png` | Primary repository and product logo |
| `logo/noval-logo-color-dark.png` | Compact color logo on dark backgrounds |
| `logo/noval-logo-color-light.png` | Compact color logo on light backgrounds |
| `logo/noval-logo-monochrome-dark.png` | White monochrome logo on dark backgrounds |
| `logo/noval-logo-monochrome-light.png` | Black monochrome logo on light backgrounds |

## Icon

| File | Intended use |
|---|---|
| `icon/noval-app-icon.png` | App-style icon with dark rounded tile |
| `icon/noval-icon-mark.png` | Standalone mark on a light background |

All selected files are opaque PNGs. They use a 32-bit ARGB pixel format, but
their canvas corners have alpha `255`; do not treat them as transparent assets.
